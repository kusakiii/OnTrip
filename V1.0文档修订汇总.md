## 三、技术方案关键修订

### 新增 DTO 校验层

```
AI 原始响应 → JSON 解析 → Schema 校验 → 自动修复 → 前端
                              ↓ 失败
                         重试 1 次 → 降级模板
```

### Schema 修正

- `transportToNext` → `transportFromPrevious`（统一语义方向）
- 数据示例中的 `styleTags: ["美食"]` → `["food"]`（统一标识符）

### 风险表更新

- **移除**：小程序拖拽交互不流畅（V1.0 已移除拖拽）
- **新增**：大模型 JSON 输出格式异常（概率中/影响高）
- **新增**：多模型 fallback 时 prompt 不兼容（概率中/影响中）
